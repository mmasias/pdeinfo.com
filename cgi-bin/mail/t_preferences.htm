Number of messages per page:
Remove from server:
Deletes messages from the incoming server as they are downloaded to MailMan.
(Do not set this if you would like to be able to download your messages from a traditional mail client even after MailMan has downloaded them.)
Remove on demand:
Deletes messages from the incoming server after they have been deleted from MailMan. If "Remove from server" is checked, this will have no effect because the messages will have been deleted from the incoming server already before they can be deleted from MailMan.
Save deleted messages:
Copies deleted messages to the 'TRASH' folder.
Save outgoing messages:
Stores a copy of outgoing messages to the 'SENT' folder.
Startup:
Go to 'INBOX':
Show message list for 'INBOX' folder on startup.
Go to folder list:
Go directly to the folder list on startup. Useful if you have multiple accounts configured to deposit mail in different folders.
